Two Side Projects

Project Description

Using C, I've written a small collection of tools to do simple tasks such as find the GCD of two numbers, solve a system of linear diophantine equations, determine if a number is prime, and find a composite number's prime factorization. This was a fun tool to work with posix threads.


I wrote the Facebook Parser in Java to parse relational information from html so that it could be used in visual analytics. Facebook provides data downloads to its users in the form of HTML documents and I am very interested in what individuals can learn about themselves from their own data. I decided to write a stand alone API rather than utilize the Facebook graph for three main reasons. In order to gain relational information from the Facebook graph one would need everyone to install their app and I was more interested in what insights into my own behavior I could gain, so I didn't want to have to depend on my all of my friends allowing my program access to their profiles. Thirdly, I wanted a Java project that would teach me more about modular design so that I could add to it over time. One example of this is being able to use jchart to plot my behavior. Ultimately, I think it would be fun to try to correlate trends to events in a person's calendar.
